When your code is compiled, the compiler needs to work out what each and every identifier in your code means. A "Cannot find symbol" error means that the compiler cannot do this. Your code appears to be referring to something that the compiler doesn\'t understand.As a first order, there is only one cause. The compiler looked in all of the places where the identifier should be defined, and it couldn\'t find the definition. This could be caused by a number of things.
