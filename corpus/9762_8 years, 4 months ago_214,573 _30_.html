Like you say, it\'s a race condition and there\'s no need. mysqli_insert_id() already has this functionality.The MySQL function LAST_INSERT_ID() does just what you need: it retrieves the id that was inserted during this session. So it is safe to use, even if there are other processes (other people calling the exact same script, for example) inserting values into the same table.The PHP function mysql_insert_id() does the same as calling SELECT LAST_INSERT_ID() with mysql_query().As to PHP\'s website, mysql_insert_id is now deprecated and we must use PDO.
