If you want to compile but not run tests use If you are using eclipse there is a "Skip Tests" checkbox on the configuration page.Run configurations \xe2\x86\x92\n Maven Build \xe2\x86\x92\n New \xe2\x86\x92\n Main tab \xe2\x86\x92\n Skip Tests\n mvn clean install -DskipTests=truemvn clean install -Dskiptests=trueIf you call your classes tests Maven seems to run them automatically, at least they did for me.
